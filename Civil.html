About?
Welcome to this Digitalize Civil status registry Project which aims to automate the process of maintaining and tracking civil status registries, as well as issuing birth and marriage certificates. This project is particularly relevant for local governments, as it helps improve the tracking and maintenance of important documents.

It allows the users to select either birth or marriage certificate options. The application can collect and store relevant data, It also has the capability to generate and print PDFs of both certificates.
